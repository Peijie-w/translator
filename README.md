# Ubersetzer

一个可直接加载到 Chrome / Edge 的浏览器扩展原型：

- 普通网页中，鼠标长时间悬停在单词上会显示翻译结果
- 通过扩展内置的 PDF Viewer 打开 PDF 后，也可以对 PDF 文本执行同样的悬停翻译
- 目标语言、源语言、悬停延迟、PDF 默认缩放都可以在设置页调整

## 当前能力

- 普通网页：支持
- 文本型 PDF：支持
- 浏览器原生 PDF 标签页：由于浏览器权限限制，不能直接注入悬停层，所以需要用扩展自带 Viewer 打开
- 扫描版 / 图片型 PDF OCR：这一版还没有接入 OCR

## 安装与构建

```bash
npm install
npm run build
```

构建完成后，产物在 `dist/`：

1. 打开 Chrome 或 Edge 的扩展管理页
2. 开启“开发者模式”
3. 选择“加载已解压的扩展程序”
4. 选中当前项目下的 `dist` 目录

## 使用方式

### 网页悬停翻译

打开任意网页，把鼠标停留在一个词上，等待默认约 700ms，就会出现翻译浮层。

### PDF 翻译

有两种方式：

1. 打开一个 PDF 链接后，点击扩展弹窗里的 `Open current PDF in viewer`
2. 在 PDF 链接上右键，选择 `Open linked PDF in Ubersetzer`

如果是本地 PDF，也可以直接打开扩展自带 Viewer，然后通过文件选择器导入。

## 翻译服务

当前版本默认通过 Google 的网页翻译接口获取结果，适合做原型验证。若你要长期使用，建议下一步切换为你自己的正式翻译 API。
